# Salesforce Validation Rule Manager - Frontend

A React web application to manage Salesforce Validation Rules.

## Live Demo
🔗 [https://salesforce-validator.vercel.app](https://salesforce-validator.vercel.app)

## Features
- ✅ Login with Salesforce (OAuth 2.0 PKCE Flow)
- ✅ Fetch all Validation Rules from Account object
- ✅ View Active/Inactive status of each rule
- ✅ Toggle single validation rule (Activate/Deactivate)
- ✅ Enable All / Disable All rules
- ✅ Logout

## Tech Stack
- React.js
- React Router DOM
- Salesforce OAuth 2.0
- Vercel (Deployment)
